# SkillProof
SkillProof is an AI + Blockchain-based platform that validates skills through interactive AI evaluation and generates dynamic, confidence-based resumes. Built during a 72-hour hackathon (LNMHacks8), it reimagines how learning and credibility are represented in the digital world through transparent, verification-driven flows.
